Encapsulation

Hiding internal details and showing only what's necessary.

It means wrapping data (variables) and code (methods) together into a single unit (class).

It protects the data by making variables private and accessing them via getters/setters.

class Person { private String name; // data hidden public void setName(String n) { name = n; } public String getName() { return name; } }
