Object Oriented Programming (OOP)

Learn OOP with real world examples

What are the main OOP concepts?
  • Encapsulation
  • Abstraction
  • Inheritance
  • Polymorphism

Encapsulation

Wrapping data and methods into a single unit, usually a class.

Access modifiers (like private) hide internal data from outside. 

class Person {
  private String name;
  public void setName(String n) { name = n; }
  public String getName() { return name; }
}

Abstraction

Hiding complex details and showing only essential features.

abstract class Animal {
  abstract void makeSound();
}
class Dog extends Animal {
  void makeSound() {
    System.out.println("Woof");
  }
}

Inheritance

Allows one class to acquire the properties of another class.

class Animal {
  void eat() { System.out.println("eating..."); }
}
class Dog extends Animal {
  void bark() { System.out.println("barking..."); }
}

Polymorphism

Allows the same method to behave differently in different classes. 

class Animal {
  void sound() { System.out.println("Some sound"); }
}
class Dog extends Animal {
  void sound() { System.out.println("Bark"); }
}
